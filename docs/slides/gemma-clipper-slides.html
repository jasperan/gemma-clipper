The Problem

Video clipping tools are stuck behind API paywalls

Cloud Lock-in
Every smart editing tool ships your video to OpenAI, Anthropic, or Google. Your content leaves your machine.
Cost Per Clip
API calls for video analysis add up fast. A 10-minute video can cost $2-5 per run. That's unsustainable for regular creators.
Privacy
Unreleased content, sensitive footage, client work. Sending it to third-party APIs isn't always an option.
The Solution

Your GPU does the thinking

gemma-clipper runs Gemma 4 (4B multimodal) locally via vLLM. It watches your video, scores every scene for engagement, and exports the best clips automatically.

  • Feed it an MP4 or YouTube URL
  • AI ranks scenes by engagement potential
  • Export clips in any format, any aspect ratio
  • Auto-captions via Whisper
  • Everything stays on your hardware
ZERO API KEYS REQUIRED
Gemma 4 E4B-it runs on a single A10 GPU (24GB) or any card with 6GB+ VRAM

Section One

Video
Input

Two ways in: drop a file or paste a URL. The pipeline handles both identically after download.

Local Files

Any MP4, WebM, MKV, AVI, or MOV

Upload via the web UI (drag-and-drop) or point the CLI at a local path. ffprobe extracts metadata: duration, resolution, fps, codec, audio tracks.

gemma-clipper analyze my_video.mp4 \ --max-clips 5 \ --max-duration 60
YouTube

Paste a URL, yt-dlp handles the rest

Automatic download with resolution capping. Extracts chapter metadata when available for chapter-aware scene boundaries. Cookie support for authenticated downloads.

gemma-clipper analyze \ "https://youtube.com/watch?v=..." \ --max-clips 10

Section Two

Scene
Detection

ffmpeg does the heavy lifting. No GPU needed for this stage. Fast enough to process a 10-minute video in seconds.

How It Works

Two detection modes, combinable

Scene Filter
gt(scene, threshold)
ffmpeg's built-in scene change detector. Compares consecutive frames, fires when the difference exceeds a configurable threshold (default 0.3). Returns timestamps + magnitude scores.
Keyframe Snapping
detect_scenes_with_keyframes()
Merges scene boundaries with codec keyframe positions. Snaps each boundary to the nearest I-frame for clean cuts without re-encoding artifacts.

Audio Analysis

Silence detection and speech mapping

The silence detector uses ffmpeg's silencedetect filter. Speech regions are the inverse: everywhere that isn't silent. Both feed into the ranking algorithm as signals.

Silence Removal
Jump cut automation
Builds a complex concat filtergraph that splices out silent segments with configurable padding. One command strips all dead air from a video.
remove_silence( source, output, threshold_db=-30.0, min_duration=0.5, padding=0.1 )
03
Section Three

Gemma 4
Analysis

The multimodal brain. Each video chunk gets sent to Gemma 4 E4B-it for structured analysis. No cloud. Just your GPU.

Model Pipeline

Chunk, encode, analyze, parse

Step 1
Extract Segment
ffmpeg cuts each scene at 480px, CRF 30. Keeps files under 18MB for the model context window.
Step 2
Base64 Encode
Video bytes become a data URI. Sent as a multimodal message to vLLM's OpenAI-compatible endpoint.
Step 3
Gemma Analyzes
Returns structured JSON: description, objects, mood, energy level, audio type, interest score, tags.
Step 4
Retry + Parse
Robust JSON extraction handles markdown fences, surrounding text. 2 retries on malformed output.
Scoring

4-signal weighted ranking

Each scene gets a composite score from 4 independently measured signals. The weights are tuned for social media engagement.

Clip Selection
Greedy non-overlapping
Sorts scenes by final score. Picks the highest, skips anything overlapping by more than 50%. Merges adjacent high-scorers. Respects min/max duration constraints.

Section Four

Export
Options

Every clip is configurable. Format, resolution, aspect ratio, captions, quality. One pipeline, infinite output variations.

Formats

Three output formats, four aspect ratios

MP4 (default)
H.264 + AAC. Universal playback. Configurable CRF (0-51).
WebM
VP9 + Opus. Web-optimized. Smaller files at equivalent quality.
GIF
Two-pass palettegen for quality. Great for previews and social embeds.
16:9
Landscape
9:16
Portrait / Reels
1:1
Square
orig
Keep source
9:16 uses center-crop from 16:9 source. 1:1 takes a square center cut. No letterboxing.
Captions

Whisper-powered auto-captions

faster-whisper handles transcription. Output as SRT, VTT, or burned directly into the video with 3 visual styles.

Default: white + black outline Bold: large yellow text Minimal: small gray text
# Generate subtitles gemma-clipper captions video.mp4 \ --model base --format srt # Burn captions into video gemma-clipper captions video.mp4 \ --burn --style bold # Export with captions baked in gemma-clipper export video.mp4 \ --captions --caption-style minimal \ --aspect-ratio 9:16
